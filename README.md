## PROJECTS
- **Café Order System:** This project is a Java-based application that allows students to register and log into a campus cafeteria system. It provides an interactive menu where users can add food items to a cart, apply discounts, and schedule pickups.
- **Student Enrolment System:** This is a Client-Server student enrolment system that uses a Java Swing GUI to manage student records and course registrations through a networked connection. It utilizes a centralized Derby database and Data Access Objects (DAOs) to allow administrators to perform CRUD operations while students can log in to view and enroll in specific courses.
- **Smart_Parking_System:** A Smart Parking System is a modern, technology-driven solution designed (using Java) to automate the process of managing parking spaces, vehicles, and attendants. Instead of manual ticketing and physical searching for spots, it uses software architecture to streamline operations.
